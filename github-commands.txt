# GitHub Repository Setup Commands

# 1. Create remote repository
gh repo create smart-forward --public --description "Smart Forward - Network forwarding tool"

# 2. Add remote origin
git remote add origin https://github.com/cls44/smart-forward.git

# 3. Push code
git branch -M main
git push -u origin main

# 4. Create first tag
git tag v1.0.0
git push origin v1.0.0

# 5. View Actions
gh run list
